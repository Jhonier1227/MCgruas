Servicios recientes

Galería visual para mostrar las imágenes más recientes del servicio

Esta galería se administra desde una carpeta exclusiva y un JSON muy simple. Solo agregas o borras imágenes en assets/servicios-recientes/ y registras número, nombre y fecha en el JSON.

Cómo mantener esta galería

Flujo simple para subir imágenes nuevas

1. Sube la imagen

Guarda la foto nueva dentro de assets/servicios-recientes/.

2. Registra el JSON

En el archivo nuevo solo colocas número, nombre del archivo y fecha del servicio.

3. Borra o reemplaza fácil

Si una imagen ya no se usará, la eliminas de la carpeta y del JSON sin tocar el HTML.
